# Demo-Taro
## 技术
Taro + JavaScript + scss + Taro UI +Vant Weapp
## 项目运行
### 全局安装Taro脚手架
npm install -g @tarojs/cli
### 安装项目依赖
npm install
### 运行（微信开发者工具）
npm run dev:weapp
或
yarn dev:weapp
